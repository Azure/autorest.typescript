# Compute client library for JavaScript

This package contains an isomorphic SDK (runs both in Node.js and in browsers) for Compute client.

Compute Client

Key links:

- [Package (NPM)](https://www.npmjs.com/package/@msinternal/compute)

## Getting started

### Currently supported environments

- [LTS versions of Node.js](https://github.com/nodejs/release#release-schedule)
- Latest versions of Safari, Chrome, Edge and Firefox.

See our [support policy](https://github.com/Azure/azure-sdk-for-js/blob/main/SUPPORT.md) for more details.


### Install the `@msinternal/compute` package

Install the Compute client library for JavaScript with `npm`:

```bash
npm install @msinternal/compute
```



### JavaScript Bundle
To use this client library in the browser, first you need to use a bundler. For details on how to do this, please refer to our [bundling documentation](https://aka.ms/AzureSDKBundling).

## Key concepts

### ComputeClient

`ComputeClient` is the primary interface for developers using the Compute client library. Explore the methods on this client object to understand the different features of the Compute service that you can access.
