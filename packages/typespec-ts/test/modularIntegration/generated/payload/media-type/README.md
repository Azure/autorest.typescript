# MediaType client library for JavaScript

This package contains an isomorphic SDK (runs both in Node.js and in browsers) for MediaType client.

Test the payload with different media types and different types of the payload itself.

[Package (NPM)](https://www.npmjs.com/package/@msinternal/payload-mediaType) |

## Getting started

### Currently supported environments

- [LTS versions of Node.js](https://github.com/nodejs/release#release-schedule)
- Latest versions of Safari, Chrome, Edge and Firefox.

See our [support policy](https://github.com/Azure/azure-sdk-for-js/blob/main/SUPPORT.md) for more details.





### JavaScript Bundle
To use this client library in the browser, first you need to use a bundler. For details on how to do this, please refer to our [bundling documentation](https://aka.ms/AzureSDKBundling).

## Key concepts

### MediaTypeClient

`MediaTypeClient` is the primary interface for developers using the MediaType client library. Explore the methods on this client object to understand the different features of the MediaType service that you can access.
