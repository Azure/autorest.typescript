# NotDefined client library for JavaScript

This package contains an isomorphic SDK (runs both in Node.js and in browsers) for NotDefined client.

Illustrates server doesn't define endpoint. Client should automatically add an endpoint to let user pass in.

[Package (NPM)](https://www.npmjs.com/package/@msinternal/notdefinedparam) |

## Getting started

### Currently supported environments

- [LTS versions of Node.js](https://github.com/nodejs/release#release-schedule)

## Key concepts

### NotDefinedClient

`NotDefinedClient` is the primary interface for developers using the NotDefined client library. Explore the methods on this client object to understand the different features of the NotDefined service that you can access.
